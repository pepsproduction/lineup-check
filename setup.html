ตั้งค่าระบบ

เชื่อมต่อกับ Google Apps Script ครั้งแรก
ทำเพียงครั้งเดียวต่ออุปกรณ์

1
สร้าง Google Sheet
  1. ไปที่ sheets.google.com
  2. สร้าง Spreadsheet ใหม่ ตั้งชื่อว่า "LineUp Check DB"
  3. จำ Spreadsheet ID ไว้ (อยู่ใน URL)
2
ติดตั้ง Google Apps Script
  1. เปิด Google Sheet ที่สร้างไว้
  2. เลือกเมนู Extensions → Apps Script
  3. ลบโค้ดเดิม แล้ววางโค้ดจากไฟล์ apps-script/Code.gs
  4. แก้ไข SPREADSHEET_ID ในโค้ดให้ตรงกับ Sheet ของคุณ
  5. กด Save (Ctrl+S)
3
Deploy เป็น Web App
  1. กด Deploy → New deployment
  2. เลือก Type: Web App
  3. Execute as: Me
  4. Who has access: Anyone
  5. กด Deploy แล้วคัดลอก Web App URL
4
ใส่ Web App URL
URL จะเริ่มต้นด้วย https://script.google.com/macros/s/
5
สร้างข้อมูลเริ่มต้น

กดปุ่มด้านล่างเพื่อให้ Apps Script สร้าง Sheet และ Admin account เริ่มต้น

⚠️ ทำเพียงครั้งเดียว — ถ้าทำซ้ำข้อมูลจะไม่ถูกลบ
